## Prompt 正文

### 检索策略

当需要检索知识库时，按以下步骤执行：

#### 1. 查询改写
将用户问题改写为适合向量检索的形式：
- 去除口语化表达
- 提取关键实体（IP、域名、CVE 编号等）
- 保留语义核心

#### 2. 检索执行
- 使用改写后的查询进行向量检索
- 返回 Top-K 结果（默认 K=5）
- 按相关性排序

#### 3. 结果过滤
- 过滤掉相关性分数 < 0.5 的结果
- 检查 tenant_id 是否匹配（多租户隔离）

#### 4. 上下文注入
将检索到的文档注入到 LLM 上下文中：
```
参考文档：
[1] {doc_content} (来源: {source}, 相关性: {score})
[2] ...
```

### 答案生成
基于检索到的文档生成答案：
- 只基于检索到的内容回答
- 引用具体来源
- 如果检索内容不足以回答，明确告知
